# 👋 Sobre mim

Sempre fui uma pessoa curiosa e gosto de entender como as coisas funcionam. Desde criança, criar algo com as próprias mãos sempre me motivou, e foi no desenvolvimento de software que encontrei um ambiente onde criatividade, lógica e resolução de problemas caminham juntas.

Ao longo da minha trajetória busquei conhecer diferentes áreas da tecnologia. Meu primeiro contato foi através de cursos de Arduino e, desde então, explorei diversos assuntos como Hardware, Redes de Computadores, Desenvolvimento Web, WordPress, Python, Lógica de Programação e outras áreas que contribuíram para ampliar minha visão sobre a tecnologia.

Hoje, como graduado em **Análise e Desenvolvimento de Sistemas** e estudante de **Engenharia de Software**, continuo investindo no aprendizado constante, sempre buscando evoluir através da prática.

Mais do que aprender novas tecnologias, gosto de entender como elas funcionam e como podem ser utilizadas para criar soluções reais.

---

# 🚀 Construindo para Evoluir

Acredito que o verdadeiro aprendizado acontece quando colocamos a mão na massa.

Por isso, criei um desafio pessoal de desenvolver **20 projetos**, cada um com um tema completamente diferente.

Para tornar essa experiência ainda mais próxima da realidade do mercado, utilizo uma Inteligência Artificial para simular um cliente real. Em cada projeto recebo um briefing contendo objetivos, identidade visual, requisitos e funcionalidades específicas.

Essa abordagem me permite praticar muito mais do que programação.

Durante cada projeto também desenvolvo habilidades como:

- Interpretação de requisitos;
- Organização e estruturação de código;
- Planejamento;
- Boas práticas de desenvolvimento;
- Versionamento com Git e GitHub;
- Resolução de problemas.

Cada projeto representa uma nova oportunidade de aprender algo que eu ainda não sabia.

---

# 💡 Minha forma de aprender

A teoria possui seu valor, mas é na prática que realmente aprendo.

Não tenho medo de errar.

Pelo contrário.

Cada erro representa uma oportunidade para entender melhor o problema e encontrar uma solução.

É justamente isso que mais me motiva no desenvolvimento de software.

Sempre existe um novo desafio, uma nova lógica para entender ou uma nova forma de construir algo melhor do que antes.
